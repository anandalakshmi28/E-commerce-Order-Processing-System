E-commerce Order Processing System

A backend application built using Java Spring Boot to manage order processing in an e-commerce system.
It supports order creation, tracking, status updates, and cancellation with proper business rules.

Setup & Run

1. Clone the repository
git clone https://github.com/anandalakshmi28/E-commerce-Order-Processing-System.git
cd E-commerce-Order-Processing-System

2. Build the project
mvn clean install

3. Run the application
mvn spring-boot:run

Application will start at:

http://localhost:8080
